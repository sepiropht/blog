@levelsio résume exactement pourquoi j'ai fait cet outil.

L'interface web de Kimi le fait déjà nativement (si on sait où chercher), Claude aussi, mais seulement sur le plan Max. Gemini et Grok, à ma connaissance, ne le font pas.

J'ai donc fait llm-web : une seule page, pensée mobile, qui donne accès à tous les agents de code qui tournent sur ta machine Claude Code, Kimi, Grok, Codex, Gemini. Tu parcours toutes tes anciennes sessions, tu en reprends une, ou tu démarres une nouvelle conversation, directement depuis ton téléphone. Pas de SSH, pas besoin de plisser les yeux sur une police de terminal.

[github.com/sepiropht/llm-web](https://github.com/sepiropht/llm-web)
